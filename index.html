what are the difference between flexbox and grid?

Flexbox is designed for a one-dimensional layout. We can work either row or column with flexbox. Grid is designed for a two-dimensional layout. We can work on rows and columns at the same time.

what are the difference between bootstrap and tailwind?

Bootstrap is a component-based frameworkthis pre-defined component makes responsive website development faster and easier. Tailwind is a utility-based framework. These utility classes help developers two make the user interfaces rapidly.

what is css box model?

Every element that can be displayed on a web page is comprised of one or more rectangular boxes. CSS box model typically describes how these rectangular boxes are laid out on a web page. These boxes can have different properties and can interact with each other in different ways, but every box has a content area and optional surrounding padding, border, and margin areas.

what is a semantic tag?

The core characteristic of a semantic element is that it clearly communicated its meaning to both the developer and the browser. These elements clearly define its content. The Semantic Tags are header,footer,nav,article,section,aside.
